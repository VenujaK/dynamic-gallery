
# Dynamic Gallery

Dynamic Gallery for local artists to showcase their work


## Instructions

1. Download the zip file
2. Go to wordpress > plugins > add new
3. Upload the plugin
4. Using the left hand side menu go to artworks and add your    artwork
5. Enjoy
